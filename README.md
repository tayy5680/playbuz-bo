# bo

GPG Back Office

## Project setup
```bash
yarn setup
```

### Compiles and hot-reloads for development
```bash
yarn serve
```

### Compiles and minifies for production
```bash
yarn build
```

### Lints and fixes files
```bash
yarn lint
```

### .env example
```
DEV_SERVER_HOST = localhost
VUE_APP_API_PROXY = true
VUE_APP_API_URL = https://somewhere.api
VUE_APP_FRONT_URL = https://somewhere.com
VUE_APP_PROMOTION_URL = https://somewhere.com
```
